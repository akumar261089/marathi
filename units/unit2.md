## ЁЯУШ Lesson 6: Third Person (рддреЛ / рддреА / рддреЗ)

**(Using Pronouns and Names)**

---

### ЁЯУЦ Explanation

**Third person** means **the person, animal, or thing we talk about**.

In Marathi, **names and third-person pronouns follow the SAME rules**.

ЁЯСЙ **Rule:**

> **Name = рддреЛ / рддреА / рддреЗ**

If we can replace a name with **рддреЛ / рддреА / рддреЗ**, the sentence is correct.

---

### ЁЯФ╣ Third-Person Pronouns

| Hindi           | Marathi |
| --------------- | ------- |
| рд╡рд╣ (male)       | рддреЛ      |
| рд╡рд╣ (female)     | рддреА      |
| рдпрд╣ / рд╡рд╣ (thing) | рддреЗ      |

---

### ЁЯФ╣ Names behave like Third Person

| Name / Thing | Pronoun |
| ------------ | ------- |
| рд░рд╛рдо          | рддреЛ      |
| рдЕрдорд┐рдд         | рддреЛ      |
| рд╕реАрддрд╛         | рддреА      |
| рдкреВрдЬрд╛         | рддреА      |
| рдЭрд╛рдб          | рддреЗ      |
| рдкреБрд╕реНрддрдХ       | рддреЗ      |

---

### ЁЯФ╣ Examples

- рд╡рд╣ рдЦреЗрд▓рддрд╛ рд╣реИ тЖТ **рддреЛ рдЦреЗрд│рддреЛ.**
- рд╡рд╣ рдЦреЗрд▓рддреА рд╣реИ тЖТ **рддреА рдЦреЗрд│рддреЗ.**
- рд░рд╛рдо рдЦреЗрд▓рддрд╛ рд╣реИ тЖТ **рд░рд╛рдо рдЦреЗрд│рддреЛ.**
- рд╕реАрддрд╛ рдЦреЗрд▓рддреА рд╣реИ тЖТ **рд╕реАрддрд╛ рдЦреЗрд│рддреЗ.**
- рдпрд╣ рдкреЗрдбрд╝ рд╣реИ тЖТ **рддреЗ рдЭрд╛рдб рдЖрд╣реЗ.**

ЁЯСЙ Notice:
Verb changes **because of gender**, not because of the name.
